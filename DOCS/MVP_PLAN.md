# CIVICLENS AI MVP PLAN

## Goal
Build an AI-powered grievance intake and routing system.

## MVP Features
- Complaint submission
- Complaint storage
- AI-based category prediction
- Urgency prediction
- Department routing
- Dashboard stats
- Complaint status updates

## Demo Flow
1. User submits complaint
2. System analyzes complaint
3. Complaint is stored with AI fields
4. Admin views dashboard
5. Admin updates complaint status
